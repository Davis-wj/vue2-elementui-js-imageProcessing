# image-peocessing
## 项目介绍

本项目，为了完成毕设设计。项目中使用了vue2+element-ui+js来实现相关植物叶片面积的计算。

## 项目安装
```
yarn install
```
或者
```
npm i
```

### 在开发环境进行编译和热加载使用
```
yarn run serve
```
或者
```
npm run serve
```

### 在生产环境中编译使用
```
yarn run build
```

### 运行你的 tests
```
yarn run test
```

### 检查修复你的文件
```
yarn run lint
```
